The position Property (3 Questions)

4. Which value of position would you use to make an element always appear in the same place within the browser window?
Correct! This is often used with popup windows to make sure they are visible whenever they are open.
5. Which value of position initially places elements according to the normal document flow but allows you to reposition the element with respect to itself, using the top, left, bottom, and right properties?
Actually, the correct answer is relative. A relatively positioned element is initially placed according to the normal document flow, and although the element can be offset from this initial position, its neighbors will behave as if it hadn't been moved.
6. True or false. Absolutely positioned elements are positioned with respect to their immediate parent container.
Actually, this is only sometimes true. An absolutely positioned element will be positioned with respect to its closest ancestor that is not statically positioned. If all of its ancestors are statically positioned, it will be positioned with respect to the viewport.
